# Operators in JavaScript 
## Arithmetic Operator
- `*` : Multiplication
- `/` : Division Operator => *Returns Quotient*
- `%` : Modular Operator => *Return Reminder*

<hr>

### Increment Operator
`++` : This increase the value by 1 
```javascript
let counter = 0 ;
counter++ // This is prefix
++counter // This is suffix
```
<hr>
### Decrement Operator 
`--` : This decrease the value by 1

```javascript
let couter = 0 ;
couter--;
--counter;
```
<hr>


### Comparison Operator
`<, <=,>= ,> , ==` Helps to compare 2 oprands 
```javascript
if(10<=20) // Means 10 is less than or equal to 20
if(10>=20) // Means 10 is greater than or equal to 20
if(10>20) // Means 10 is greater than  20
if(10<20) // Means 10 is less than  20
if(10==20) // Means 10 is equal to  20
if(10!=20) // Means 10 is not-equal  to  20

// Strict Equality 
if(10===20) // Means 10 is equal to  20 (tripple equal to check datatype and value both)

// Strict Un-Equality
if(10!==20) // Means 10 is equal to  20 (tripple  not-equal to check datatype and value both) 
```

<u>__Note__ : _Try to alway use Strict Equality and Strict Un-Equality_ </u>
